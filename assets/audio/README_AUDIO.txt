Meowmoon Bowling v0.1 audio notes

1. Background music
   The game first tries to load:

     assets/audio/bach-placeholder.mp3

   Put the Bach MP3 at that exact path/name to use it. If the file is absent, the game falls back to a soft generated test loop so the prototype remains playable.

2. Sound effects
   v0.1 uses in-code Web Audio placeholder effects for rolling, pin hit, pin fall, and reward. These are generated by the game code and are not third-party recordings.

3. Candidate external sound sources reviewed for a later replacement pass
   - OpenGameArt MySFX page includes a listed "Bowling Ball Rolling" item; page located at https://opengameart.org/content/mysfx
   - OpenGameArt "75 CC0 breaking / falling / hit sfx" page located at https://opengameart.org/content/75-cc0-breaking-falling-hit-sfx
   - OpenGameArt CC0 Sounds Library page located at https://opengameart.org/content/cc0-sounds-library

Before bundling outside sounds in a release, download the exact files, preserve the downloaded license/attribution text, and confirm that each selected file is CC0 or otherwise acceptable for the intended game distribution.
